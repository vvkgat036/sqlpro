## 核心功能一览

不用堆砌功能点，挑几个日常真正会用到的说：

**语法高亮 + 智能补全**：写 SQL 的时候会自动补全关键词和表名，还支持多种深色主题，眼睛不容易累。

**多查询并行执行**：可以同时跑多条查询，结果分 Tab 显示，包括报错信息和消息。不用跑完一条再跑下一条，效率翻倍。

**像 Excel 一样编辑数据**：可以直接选中单个单元格进行编辑，不用像某些工具那样必须选整行才能改。对经常手动调数据的同学来说，这个体验差别很明显。

**云数据库支持**：Amazon RDS、Microsoft Azure、Heroku 都能直接连，还支持 SSH 隧道，远程访问安全有保障。

**主密码保护**：对存储的数据库连接信息加一层主密码，稍微安心一点。
